# Guessing Game 🎯

A simple **Java console game** where the player tries to guess a randomly generated number between 1 and 100.  
This game is perfect for practicing **basic Java programming concepts** like input/output, loops, conditionals, and random number generation.

---

## How to Play

1. The program generates a random number between **1 and 100**.  
2. Enter your guess when prompted.  
3. The game will tell you if your guess is **too high**, **too low**, or **correct**.  
4. Keep guessing until you find the correct number.  
5. The program counts your attempts and congratulates you when you win.

---

## How to Run

1. Make sure **Java is installed** on your machine.  
2. Open a terminal in the folder containing `GuessingGame.java`.  
3. Compile and run the game:
```console
javac GuessingGame.java
java GuessingGame
```

---

```console

Congratulations! You guessed the number in 3 attempts.
```

## Learning Outcomes

By playing and reviewing this game, you will strengthen your understanding of:

- Java input/output using `Scanner`
- Random number generation using `Random`
- Loops (`do-while`)
- Conditional statements (`if-else`)
